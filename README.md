# qnote

A lightweight X11-based note-taking tool.

## Features
- Fast and minimal.
- Graphical interface using X11 and Xft.
- Custom hotkeys support.

## Dependencies
You will need the following libraries to build `qnote`:
- `libx11-dev`
- `libxft-dev`
- `pkg-config`

## Building
Run the following command in the project directory:
```bash
make
```

## Installation
To install `qnote` to your system:
```bash
sudo make install
```

## Usage
Simply run `qnote` from your terminal or binder.
